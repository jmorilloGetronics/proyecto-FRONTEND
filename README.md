# Concesionario Leo - Frontend
Este proyecto es la interfaz de usuario para el catálogo de coches.
- **Tecnología:** HTML5, CSS3, JavaScript (Fetch API).
- **Conexión:** Consume la API REST del backend en `http://localhost:8080/api/coches`.

## Cómo ejecutar
1. Tener el backend arrancado.
2. Abrir `index.html` en el navegador.
